# Legal Olympiad Student Dashboard

This project is a student dashboard designed for the Legal Olympiad platform. It provides a structured and clear view of student progress, rankings, and achievements.

---

## 🚀 Features

- Dashboard summary cards (Internships, Courses, Marks, Rankings)
- Progress tracking using a progress bar
- Data visualization using charts (Line and Bar)
- Sidebar navigation for better user experience
- Recent activity section
- Clean and responsive UI

---

## 🛠️ Tech Stack

- React + Vite
- Tailwind CSS
- Chart.js

---

## 🧠 Approach

I focused on creating a clean and user-friendly dashboard. The layout is structured with a sidebar and main content area to ensure easy navigation.

I used reusable components for cards and charts, and maintained readability through proper spacing, typography, and color hierarchy. The goal was to present data in a clear and visually appealing way.

---

## ⚙️ How to Run

1. Clone the repository or download and unzip the project
2. Navigate to the project root directory using terminal or command prompt
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
